**Key findings to apply in future research:**

2. Fireflies achieved unicorn status ($1B) on "Talk to Fireflies" — real-time voice assistant with Perplexity web search during meetings (June 2025). This is the hottest market narrative.

3. Otter hit $100M ARR, launched AI Meeting Agents + SDR Agent + MCP server + HIPAA in 2025.

4. Microsoft Teams is adding bot identification for admins (rolling out May 2026). This could restrict browser-bot access.

5. Zoom RTMS SDK (2025) is the faster path to Zoom support — avoids Marketplace review delay of Meeting SDK.

6. Privacy backlash is real: Oxford blocked meeting bot SSO, Fireflies faces BIPA class-action, Recall.ai launched Desktop SDK in response.

7. Priority order for Vexa execution: Scheduler fix → Calendar Integration + Chat validation (parallel) → Knowledge workspace entity extraction → Zoom support.

8. MCP is becoming table stakes — Otter and MeetGeek both shipped MCP servers.
